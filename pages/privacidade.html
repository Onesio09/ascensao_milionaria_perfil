Política de Privacidade

Última atualização: julho de 2026

1. Quem Somos

A Ascensão Milionária ("nós", "o site") é um projeto operado por Onésio Belino. Para questões sobre esta política ou os teus dados pessoais, contacta: onesiobelino@gmail.com

2. Que Dados Recolhemos

Quando crias conta no site, recolhemos:

  • Nome
  • Endereço de email
  • Data de criação da conta
  • Progresso na Academia (se aplicável)
  • Código de afiliado, se te tornares afiliado

Também recolhemos dados de navegação de forma automática, através de cookies e tecnologias semelhantes, para fins de estatística (Firebase Analytics) e publicidade (Adsterra, Monetag).

3. Como Usamos os Teus Dados

  • Para criar e gerir a tua conta
  • Para guardar o teu progresso na Academia
  • Para gerir o programa de Afiliados
  • Para comunicar contigo sobre a tua conta ou o site
  • Para melhorar o site com base em estatísticas de utilização

4. Serviços de Terceiros

Utilizamos os seguintes serviços, que podem processar os teus dados de acordo com as suas próprias políticas:

  • Google Firebase (Authentication, Firestore, Analytics) — alojamento de conta, base de dados e estatísticas
  • Adsterra — rede de publicidade (Social Bar, Popunder)
  • Monetag — rede de publicidade (Popunder, Interstitial)
  • GitHub Pages — alojamento do site

Estes serviços podem usar cookies próprios para funcionamento e personalização de anúncios.

5. Os Teus Direitos (RGPD)

Tens direito a:

  • Aceder aos dados que temos sobre ti
  • Corrigir dados incorretos
  • Pedir a eliminação da tua conta e dados
  • Pedir a portabilidade dos teus dados
  • Opor-te ao processamento dos teus dados para fins de marketing

Para exercer qualquer um destes direitos, contacta-nos através do email indicado no ponto 1.

6. Retenção de Dados

Mantemos os teus dados enquanto a tua conta estiver ativa. Se pedires a eliminação da conta, os teus dados pessoais serão apagados dos nossos sistemas, exceto quando a lei exigir retenção por período superior.

7. Menores de Idade

O site não se destina a menores de 18 anos. Não recolhemos intencionalmente dados de menores.

8. Alterações a Esta Política

Podemos atualizar esta política periodicamente. A data da última atualização está sempre indicada no topo desta página.
